Thermodynamic Calculator

You can choose to work with the online version which has a simpler, text based UI or you can download one of the binaries for your OS.

Online

This is the online version deployed on Heroku. You can click on the following link to use the calculator.

Thermodynamic Calculator Online

Binaries

You can also download the exectuable binaries for your particular operating system.

Mac Instrucitons
  • Download the compressed binary Calculator for Mac
  • Unzip the folder in a location of your choice
  • Open a terminal and navigate to the uncompressed folder
  • Run the app by typing ./run.sh
Windows Instructions
  • Download the compressed binary Calculator for Windows
  • Unzip the folder in a location of your choice
  • Navigate to the uncompressed folder.
  • Double click on the file load_chem_app.exe
  • A terminal with the app should open.
Linux Instructions
  • Download the compressed binary Calculator for Linux
  • Unzip the folder in a location of your choice
  • Open a terminal and navigate to the uncompressed folder.
  • Run the app by typing ./run.sh
